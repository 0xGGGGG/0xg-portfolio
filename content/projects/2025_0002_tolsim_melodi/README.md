# TOLSim
# 

A small physics and immersive (MR) builder prototype to experiment with a custom joint system that one of my
kinetic architect friends developed and patented.

There is a specific type of joint and also a stick. Then we start to build up different geometrical forms.
Unreal Engine is used for XR App develoopment.

These has been prototyped to apply for a government grant in Turkey.
