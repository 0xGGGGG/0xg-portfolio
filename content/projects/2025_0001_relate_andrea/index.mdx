*Relate — Talk With Me* is an installation by **Andrea Familari** at **uzwei**
in the Dortmunder U, Dortmund — a room where visitors speak with the work and
the work answers.

I built the voice: an **NLX.ai** conversational agent wired into **Unreal Engine**
as a reusable Blueprint component, so the runtime could listen, route intent and
respond in real time inside the engine driving the piece.
